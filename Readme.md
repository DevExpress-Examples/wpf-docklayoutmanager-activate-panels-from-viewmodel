# How to activate panels via ViewModel


<p>This example shows how to activate a panel via two-way binding in the BaseLayoutItem.IsActive property. After changing an active panel, a corresponding ViewModel is activated. And after changing the active ViewModel, the active panel is passed to DockLayoutManager.<br /></p>
